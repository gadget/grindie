h2. Grindie to perform easily controlled large scale load tests

A bunch of shell and jython scripts on top of the "Grinder":http://grinder.sourceforge.net framework.

Features:
* more realistic scenarios with the possibility to run different testcases (grinder scripts) on different agent machines parallel
* control scripts and testcases can be automatically deployed on agent machines
* logs can be retrieved from all the agent machines automatically for further analysis
* quick and easy ramp up of Amazon EC2 instances
* pure bash code with no exotic dependencies

For more information please check the "wiki":https://github.com/gadget/grindie/wiki.
